## 🎯 Interview Question Mapping

### JavaScript Core → Interview Questions

```
Concept: Closures
├─ Basic: "What is a closure?"
├─ Intermediate: "Implement a function that returns a counter"
├─ Advanced: "How do closures affect memory? Create a memory leak example"
└─ Expert: "Optimize a closure-heavy codebase for performance"

Concept: Event Loop
├─ Basic: "Explain the event loop"
├─ Intermediate: "Predict the output of setTimeout vs Promise"
├─ Advanced: "How does the event loop handle different task types?"
└─ Expert: "Design a custom scheduler using event loop principles"

Concept: Prototypes
├─ Basic: "What is prototype inheritance?"
├─ Intermediate: "Implement classical inheritance with prototypes"
├─ Advanced: "Create a mixin system using prototypes"
└─ Expert: "Optimize prototype chain for performance"
```

### React → Interview Questions

```
Concept: Hooks
├─ Basic: "What are hooks and why were they introduced?"
├─ Intermediate: "Create a custom hook for API calls"
├─ Advanced: "Implement useState from scratch"
└─ Expert: "Design a hook system for complex state management"

Concept: Performance
├─ Basic: "What is React.memo?"
├─ Intermediate: "Implement virtualization for a large list"
├─ Advanced: "Profile and optimize a slow React component"
└─ Expert: "Design a performance monitoring system for React"
```

### CSS → Interview Questions

```
Concept: Layout
├─ Basic: "Explain the box model"
├─ Intermediate: "Create a responsive layout without media queries"
├─ Advanced: "Implement a masonry layout with CSS Grid"
└─ Expert: "Design a dynamic layout system with CSS variables"

Concept: Performance
├─ Basic: "What causes reflow and repaint?"
├─ Intermediate: "Optimize CSS for 60fps animations"
├─ Advanced: "Implement GPU-accelerated animations"
└─ Expert: "Design a CSS architecture for large-scale applications"
```

```javascript
const obj = { a: 1 };
const fn = function() { return this.a; };
const bound = fn.bind(obj);
const result = bound();
console.log(result); // ?

// What's the output and why?
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 100);
}

// How would you fix the above?
```

**React Patterns Test:**
```javascript
// Identify the anti-patterns and fix them
const BadComponent = () => {
  const [data, setData] = useState([]);
  
  useEffect(() => {
    fetch('/api/data')
      .then(res => res.json())
      .then(setData);
  });
  
  return (
    <div>
      {data.map(item => 
        <div key={Math.random()}>{item.name}</div>
      )}
    </div>
  );
};
```
