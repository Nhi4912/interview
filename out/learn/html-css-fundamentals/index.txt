# HTML & CSS Fundamentals

## Overview

HTML and CSS form the foundation of web development. This guide covers semantic HTML, modern CSS layout techniques, and best practices.

## Semantic HTML

### Document Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <header>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section>
            <h1>Main Heading</h1>
            <article>
                <h2>Article Title</h2>
                <p>Content here...</p>
            </article>
        </section>
    </main>
    
    <footer>
        <p>&copy; 2024 Company Name</p>
    </footer>
</body>
</html>
```

## CSS Layout

### Flexbox

```css
.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 1rem;
}

.flex-item {
    flex: 1;
}
```

### CSS Grid

```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}
```

## Responsive Design

### Mobile-First Approach

```css
/* Base styles (mobile) */
.container {
    padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        padding: 2rem;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        padding: 3rem;
    }
}
```

## Accessibility

### ARIA and Semantic Elements

```html
<nav aria-label="Main navigation">
    <ul>
        <li><a href="/" aria-current="page">Home</a></li>
        <li><a href="/about">About</a></li>
    </ul>
</nav>

```

## Best Practices

1. **Use semantic HTML elements**
2. **Design mobile-first**
3. **Optimize for accessibility**
4. **Use CSS custom properties**
5. **Keep specificity low**
